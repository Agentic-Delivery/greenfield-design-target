01 · Colour

The colour system

One clay accent under 80% saturation, carried at roughly one-tenth of the visual weight. Paper-warm neutrals tinted toward the same hue — never pure grey, never pure black or white. State colours are signal-only and always travel with an icon and a label.

02 · Type

The type system

Newsreader — an editorial serif drawn for reading on screen — sets every heading and the reading specimen. Cabin, a humanist sans, carries the interface. A modular scale (1.25) keeps each level clearly distinct.
